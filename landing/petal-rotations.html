v0 Prompt -- Copy This

Create a React component called SakuraPetals that renders falling cherry blossom petals. Each petal is this SVG path: <path d="M20 2C22 10 30 18 28 28C26 38 20 40 18 36C16 32 14 22 16 14C18 6 19 2 20 2Z" fill with a radial gradient from #FFD4DE center to #FFB7C5 edge>. The component spawns 3-5 petals. Each petal gets RANDOMIZED values on mount: startX (0-100vw), size (16-32px), fallDuration (6-12s), swayAmplitude (40-120px), rotationSpeed (0.5-2x), startDelay (0-5s), opacity (0.4-0.8). Use CSS 3D transforms: combine rotateX (0-360), rotateY (0-360), rotateZ (0-360) with translateX (sine wave sway) in a single keyframe animation. The petal should look like it's TUMBLING -- not just falling straight. At 80% progress it fades to 0. On animation end, respawn with new random values. No animation libraries, pure CSS keyframes with CSS custom properties for the random values. Add perspective: 800px on the container. The back face should be slightly darker (#FFA8B8). Use backface-visibility: visible.
